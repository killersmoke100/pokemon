## Getting Started
1) Clone the repo 
2) Navigate to the pokemon directory
3) Install dependencies 

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```
4) Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

5) Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
